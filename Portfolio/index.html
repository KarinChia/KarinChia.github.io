Projects

StudyBuddy Website

I restructured the layout using CSS, added icons, images, and a press slideshow, and organized content with divs and sections for clarity. I introduced a cohesive color scheme and responsive design elements to enhance visual appeal and usability across devices.

Prototype Health App

I used rectangle and ellipse elements to structure each section, designed custom icons with the pen tool, and carefully considered text hierarchy for clarity. I chose a violet and yellow color scheme to reflect the app's focus on sleep and well-being.

JS Dice Game

I used JavaScript to implement a function that lets users play multiple rounds in one session, using a for loop to track each round and calculate the user's balance. I also used CSS to reorganize the visual hierarchy and styled the page with playful elements to match the game's lighthearted theme.

Xenith Delta Poster

I started with thumbnail sketches in Procreate and worked with my team to select the strongest concept. I used Photoshop with layer masks and selections to refine source images, arranged elements thoughtfully to establish hierarchy, and created a custom glitch-style font to match the cyberpunk theme.
